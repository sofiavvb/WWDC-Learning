# Tags

As tags são **controladas** para manter o grafo do Obsidian limpo e conectado. Reutilizar
tags faz com que notas de assuntos parecidos se liguem no grafo; inventar tags soltas
quebra essas conexões.

Para adicionar uma tag nova: inclua-a nesta lista **no mesmo commit** da nota que a usa.

## Tags disponíveis

- `Foundation-Models`
- `testing`
- `on-device-AI`
- `evaluation`
- `AI-ML`
- `Vision-Framework
- `tool-calling`
- `image-understanding`


## Convenção

Use hifens, sem espaços (`on-device-AI`, não `on device AI`). Prefira reutilizar uma tag
existente a criar uma quase-duplicata (ex: não crie `tests` se já existe `testing`).
