## Convenção de nomes

Para qualquer imagem, use o padrão:

```
{session-slug}-{author}-{n}.png
```

- `session-slug` — o mesmo slug do arquivo da nota (ex: `meet-the-evaluations-framework`)
- `author` — seu handle (o mesmo usado no nome do arquivo da nota)
- `n` — um número sequencial (1, 2, 3, …)

O sufixo com o seu handle evita conflitos de nome quando duas pessoas anotam a mesma
sessão e ambas têm screenshots. Referencie a imagem na nota com a sintaxe de embed do
Obsidian: `![[meet-the-evaluations-framework-sofiavvb-1.png]]`.
